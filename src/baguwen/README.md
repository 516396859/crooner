## 介绍
此笔记的八股文是面向Java后端开发初学者的，在应对企业对开发者越来越高的要求的背景下，一个合格的Java后端开发者需要具备的技能不仅仅是会写代码，还需要具备一定的基础知识，这些基础知识包括Java基础、Java进阶、数据库、SSM框架、中间件、微服务、计算机基础、JVM、设计模式、中级算法等等，这些知识点是Java后端开发者的基础，也是Java后端开发者的核心竞争力。本章的优势主要是简单、易懂、全面、系统，适合Java后端开发初学者阅读。
